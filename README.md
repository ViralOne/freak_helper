# Freak Helper

## First we need to install the dependencies

```sh
apt-get update 
sudo apt-get install dialog
```

## Use the script in your terminal

```sh
chmod +x afhelper
./afhelper
```
## Install the app:

```sh
sudo install afhelper /usr/local/bin
```

### How to start the app

Open a terminal an run inside the next line:

```sh
 afhelper
```
